# 🎮 Tic Tac Toe

A simple and responsive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo
🔗 https://saloni-78.github.io/Tic-Tac-Toe/

---

## 📌 Features
- Two-player gameplay
- Responsive design for mobile and desktop
- Reset game functionality
- Clean and simple UI
- Instant win detection

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📱 Responsive Design

The project uses **CSS Grid**, **Flexbox**, and **Media Queries** to ensure the layout works well on:

- 📱 Mobile devices  
- 💻 Tablets  
- 🖥️ Desktop screens
